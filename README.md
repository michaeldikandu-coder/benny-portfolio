# Michael Dikandu - Portfolio

A clean, minimalist portfolio built with Next.js 14, featuring smooth animations and a black & white design theme.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
